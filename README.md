# SCW Portfolio Marketplace

A premium source code marketplace built with Next.js, Genkit, and Firebase.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your `.env` file with your `GEMINI_API_KEY`.
4. Run the development server: `npm run dev`.

## Features

- **Premium Marketplace UI**: Optimized for Web and Mobile project categories.
- **Admin Dashboard**: Manage projects and use AI-powered copywriting tools.
- **Demo Access**: Integrated buttons for User and Admin live demos.
- **Mobile Responsive**: Custom-built mobile layouts and navigation.

## Deployment

For detailed instructions on how to deploy this application to a VPS with your custom domain, please refer to the [DEPLOY.md](./DEPLOY.md) file.
