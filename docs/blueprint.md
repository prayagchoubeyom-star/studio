# **App Name**: SCW Portfolio

## Core Features:

- Project Showcase: Display a categorized list of projects with interactive cards, including project name, brief description, and a compelling thumbnail image.
- Detailed Project View: Provide a dedicated page for each project, featuring comprehensive descriptions, a gallery of images/videos, links to live demos, and code repositories.
- Interactive 3D Elements: Integrate interactive 3D models or scenes within selected project detail pages, allowing users to explore aspects of the project visually.
- AI Project Summarizer Tool: An internal tool for the developer to input raw project details and receive AI-generated captivating summaries and relevant keywords for their portfolio listings.
- Responsive Design: Ensure the entire website layout and elements adapt seamlessly across various devices and screen sizes, from mobile to desktop.
- Contact & About Section: A dedicated page or section providing the developer's professional background, skills, and a functional contact form for inquiries.

## Style Guidelines:

- Primary accent color: A deep blue-violet (#7B67E4) symbolizing innovation and expertise, standing out on a dark background.
- Background color: A very dark, subtle grey-violet (#17161D) providing a sophisticated and tech-forward canvas that highlights content.
- Secondary accent color: A bright, vibrant blue (#6E9CF7) for interactive elements and highlights, ensuring clear calls to action and visual interest.
- Headlines font: 'Space Grotesk' (sans-serif) for a modern, techy, and slightly futuristic feel that aligns with a coding portfolio.
- Body text font: 'Inter' (sans-serif) to ensure excellent readability and a clean, objective aesthetic across all content.
- Use minimalist, clean line icons that convey functionality and professionalism, supporting the modern '3d design' aesthetic without clutter.
- Implement a spacious, grid-based layout for project listings, emphasizing visuals and clear content hierarchy. Utilize full-bleed sections for immersive project details.
- Incorporate subtle, smooth transitions for navigation and content loading, enhancing the user experience. Use micro-interactions for buttons and hover states to add polish.
